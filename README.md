# text-utils README

文本格式化工具，右键选择功能，现有如下功能：  
1. TODO列表（快捷键：Alt + C）
2. 字符串逆序
3. 字符串转小写
4. 字符串转大写
5. 词首字母大写
6. 生成无序列表
7. 生成有序列表
8. base64编码解码
